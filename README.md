# 🚗 Driver Drowsiness and Distraction Detection System

## 📌 Project Title

**AI-Based Driver Drowsiness and Distraction Detection System Using Computer Vision and Deep Learning**

---

## 📖 Project Description

This project is an intelligent real-time monitoring system designed to detect driver drowsiness and distraction using computer vision and machine learning techniques. The system analyzes facial features such as eye closure, blinking rate, and yawning behavior through a webcam. When abnormal patterns are detected, the system generates an alert sound to warn the driver and prevent accidents.

The system improves road safety by providing early detection of fatigue and inattentiveness.

---

## 🎯 Objectives

* Detect driver drowsiness in real time

## 📦 Installation Steps

1. Clone the repository:

```
git clone https://github.com/your-username/driver-drowsiness-detection.git
```

2. Navigate to the project folder:

```
cd driver-drowsiness-detection
```

3. Install required dependencies:

```
pip install -r requirements.txt
```

---

## ▶️ How to Run the Project

Run the main file:

```
python main.py
```

The webcam will start automatically, and the system will monitor the driver's behavior.

If drowsiness or distraction is detected, an alert sound will be triggered.
